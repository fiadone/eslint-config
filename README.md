# @fiad/eslint-config

A custom ESLint Shareable Config

---

## Installation

```
npm i @fiad/eslint-config
```

## Usage


```json
// .eslintrc
{
    "extends": "@fiad"
}
```

For more information, check the [*ESLint* documentation](https://eslint.org/docs/user-guide/configuring#using-a-shareable-configuration-package).
